# 🍽️ Aadya's Restaurant — Premium UI Experience

Aadya's Restaurant is a modern, high-performance restaurant website built with cutting-edge frontend technologies. It delivers a **luxury digital dining experience** through immersive UI, smooth animations, and elegant design.

---

## 🚀 Live Demo

👉 https://your-live-link.com

---

## ✨ Features

### 🎨 UI / UX Excellence

* Premium dark-theme design
* Glassmorphism + gradient aesthetics
* Fully responsive across devices
* Luxury typography system

### ⚡ Advanced Animations

* Framer Motion powered transitions
* Scroll-based parallax effects
* Staggered content reveal
* 3D hover interactions

### 🧠 Smart Experience

* Lazy-loaded images for performance
* Intersection Observer-based animations
* Smooth micro-interactions

### 📊 Sections Included

* Hero Section (Landing)
* About Section (Story + Stats)
* Menu Showcase
* Testimonials / Pricing UI
* Contact Section
* Advanced Footer (Luxury UI)

## ⚙️ Installation

```bash
# Clone the repository
git clone https://github.com/your-username/aadyas-restaurant.git

# Navigate to project
cd aadyas-restaurant

# Install dependencies
npm install

# Start development server
npm run dev
```
